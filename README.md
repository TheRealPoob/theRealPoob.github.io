# Bio Website Draft

This repository contains a draft personal bio website for John Lee.

## Project structure
- `index.html` — homepage and introduction
- `portfolio.html` — portfolio, experience, and social link section
- `stuff.html` — additional content placeholder
- `style.css` — shared styles
- `script.js` — optional JavaScript
- `images/` — image assets

## Notes
- `.gitignore` excludes Obsidian metadata in `.obsidian/` so you can use your markdown notes without committing app settings.
- Markdown files are still available for version control; only Obsidian workspace data is ignored.
- The site uses a simple static sidebar layout.

## Reference
- LinkedIn: https://www.linkedin.com/in/john-lee-1a2359343/

## Getting started
1. Open `index.html` in a browser.
2. Edit the HTML files and styles to add your bio and portfolio content.
3. Add your Obsidian notes alongside the project if you want to keep notes and web content together.
